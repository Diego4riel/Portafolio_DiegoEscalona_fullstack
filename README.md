# Portafolio Fullstack 2026

Portafolio personal desarrollado con React, TypeScript y Vite, mostrando proyectos y experiencia como desarrollador fullstack.

## 🚀 Tecnologías

- React 19 + React Compiler
- TypeScript
- Vite
- TailwindCSS + shadcn/ui
- React Router DOM
- Framer Motion
- ESLint + Prettier

## 📦 Instalación

```bash
npm install
```

## 🖥️ Desarrollo

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 🔍 Linting

```bash
npm run lint
```

## 📁 Estructura del proyecto

```
src/
├── assets/
├── components/
│   ├── layout/
│   ├── sections/
│   ├── shared/
│   └── ui/
├── constants/
├── hooks/
├── lib/
├── pages/
├── styles/
├── types/
├── App.tsx
└── main.tsx
```

## 🌿 Estrategia de ramas

- ```\main\```: rama estable, lista para producción
- ```\develop\```: rama de desarrollo activo
